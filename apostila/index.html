Apostila de HTML e CSS

Introdução a HTML e CSS

HTML (HyperText Markup Language) e CSS (Cascading Style Sheets) são as duas linguagens fundamentais para a criação de páginas da web. Juntas, elas permitem que os desenvolvedores construam websites atraentes e funcionais.

HTML (HyperText Markup Language): HTML é a espinha dorsal da web. É uma linguagem de marcação que os desenvolvedores usam para estruturar o conteúdo de uma página da web. Em essência, o HTML fornece a estrutura e o esqueleto do site. Ele usa tags para envolver e definir elementos como cabeçalhos, parágrafos, imagens, links e muito mais. As tags são usadas para indicar ao navegador como exibir cada parte do conteúdo.

CSS (Cascading Style Sheets): CSS é responsável pelo estilo e apresentação de uma página da web. Ele permite que você controle a aparência visual de elementos HTML, como cores, fontes, tamanhos, margens, espaçamento e muito mais. Com CSS, você pode criar layouts atrativos e consistentes em todo o site.

Tags HTML

Tags HTML:

<html>: Define o início e o fim do documento HTML.

<head>: Contém informações sobre o documento, como título e links para estilos CSS.

<title>: Define o título da página, que aparece na guia do navegador.

<body>: Contém o conteúdo visível da página, como texto, imagens, links e outros elementos.

CSS

O CSS é usado para estilizar páginas web. Exemplo:

background-color: Define a cor de fundo de um elemento.

font-family: Especifica a família da fonte a ser usada.

margin: Define as margens ao redor de um elemento.

border: Define as bordas de um elemento.

Exemplos de Código

Exemplos de código HTML e CSS:

<html>
<head>
    <title>Trabalho</title>
    <link rel="stylesheet" type="text/css" href="style.css">
</head>
<body>
    <h1>Bem-vindo ao trabalho</h1>
    <p>Este é um exemplo de página web.</p>
</body>
</html>
